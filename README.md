## Frameworks
- ### UI Framework - [Laravel](https://laravel.com)
- ### Database ORM - [Eloquent ORM](https://laravel.com/docs/5.0/eloquent)
- ### Default Database - [MySQL](https://www.mysql.com/)

CMS Multi Landing Pages Company Profil adalah sebuah sistem manajemen konten (Content Management System) yang dirancang untuk membuat dan mengelola banyak landing page sekaligus. Project ini ditujukan bagi perusahaan, UMKM, organisasi, maupun individu yang membutuhkan halaman profil profesional dengan pengelolaan terpusat.

## Fitur Utama
- Multi Landing Pages  
Admin dapat menambahkan banyak landing page untuk berbagai entitas (perusahaan, UMKM, startup, dll) dalam satu sistem.
- Dashboard Admin  
Tersedia halaman dashboard yang intuitif untuk mengelola data, konten, dan tampilan setiap landing page.
- Manajemen Konten  
Mendukung pengaturan teks, gambar, video, dan elemen branding sesuai kebutuhan masing-masing landing page.
- Template Responsif  
Landing page otomatis menyesuaikan tampilan di desktop, tablet, maupun smartphone.
- Pengelolaan Data Terpusat  
Semua data landing page tersimpan dalam satu sistem sehingga mudah dipantau dan diperbarui.
- Akses Multi User  
Mendukung peran berbeda (admin, editor, viewer) untuk kolaborasi tim.

## Tujuan Project
- Mempermudah perusahaan dan UMKM dalam membuat profil online yang profesional.
- Menghemat waktu dengan sistem terpusat untuk mengelola banyak landing page.
- Memberikan fleksibilitas desain dan konten sesuai kebutuhan masing-masing entitas.
- Meningkatkan visibilitas digital melalui landing page yang modern dan responsif.

## Contoh Use Case
- Perusahaan konsultan yang memiliki banyak klien, masing-masing dengan landing page khusus.
- UMKM yang ingin menampilkan produk atau layanan berbeda dalam halaman terpisah.
- Organisasi yang butuh landing page untuk event, kampanye, atau divisi tertentu.
